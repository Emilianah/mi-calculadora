# 🧮 Mi Calculadora en React

Este proyecto es una aplicación web desarrollada con React que permite realizar operaciones básicas y navegar entre diferentes vistas.

## 🚀 Funcionalidades

* 📌 Menú principal con navegación
* ➕ Suma de dos números con resultado en alerta
* 🧮 Calculadora completa con:

  * Números del 0 al 9
  * Operaciones: suma, resta, multiplicación y división
  * Botón de igual (=)
  * Botón de borrar (C)
* 🔙 Navegación entre páginas

## 🛠️ Tecnologías utilizadas

* React
* JavaScript
* HTML
* CSS
* React Router DOM

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio:

   ```
   git clone https://github.com/Emilianah/mi-calculadora.git
   ```

2. Entrar a la carpeta:

   ```
   cd mi-calculadora
   ```

3. Instalar dependencias:

   ```
   npm install
   ```

4. Ejecutar la aplicación:

   ```
   npm start
   ```

5. Abrir en el navegador:

   ```
   http://localhost:3000
   ```
